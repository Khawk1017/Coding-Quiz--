# Coding-Quiz--
This is a simple coding quiz application designed to test your programming skills.

## Features
- Multiple choice questions
- Easy to use
- Score tracking
- Very Responsive

## How to use

1. Open the application
2.  Answer the questions 
3. Track your score over time

## Technologies Used
- HTML
- CSS
- JavaScript
